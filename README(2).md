# 🔐 CLI Password Manager

A simple command-line Password Manager application built using Python.

## 🚀 Features
- Add new account (website, username, password)
- View saved accounts (passwords hidden for security)
- Search accounts by keyword
- Data stored using file handling

## 🛠️ Technologies Used
- Python

## 💾 How It Works
All account details are stored in a file (`passwords.txt`) in the format:

website,username,password

The program reads and writes data from this file to manage accounts.

## ▶️ How to Run

```bash
python password_manager.py
```
